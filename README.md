# PBL1-2
## 📁 任务一：手部抓握运动神经活动分析 (Task 1)

本部分旨在通过时频分析方法，处理硬膜外脑电（eECoG）信号，提取患者尝试手部抓握时的神经响应特征。

* **`task1_ersp.py`**：核心代码脚本。用于读取连续的单次抓握电生理数据，进行 Epoch 切片、跨 Session 平均、计算多窗口时频表示（TFR），并最终绘制时频图与频率曲线图。
* **`Fig_1c_Time_Frequency.png`**：运行代码生成的 8 通道时频图。展示了对侧手部运动时，右脑感觉运动皮层上低频 ERD（事件相关去同步化）与高频 ERS（事件相关同步化）的时间动态演变。
* **`Fig_1d_Left_Right_Contrast.png`**：运行代码生成的左右手对比曲线图。在核心运动通道（ch3）上，严格对标文献方法，对比了对侧运动与同侧运动在不同频率下（特别是 High $\gamma$ 频段）的 ERSP 响应差异。
* **`Task1_Analysis.md`**：详细的图表分析报告。深入解读了上述图表背后的神经生理学意义，并为后续的连续解码和多手势分类任务（Task 2-4）提供了关键的**特征工程提取指南**。
